## SqliteClosedHelper


## How To Add to Your Project? 



Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.wisdomrider:SqliteClosedHelper:2.4.0'
	}
